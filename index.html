📜 History of Morse Code

Samuel Morse and Alfred Vail developed Morse code in the 1830s to send messages through telegraphs using dots and dashes. The first message was sent in 1844, and the system quickly became important for communication on ships, railways, and in the military. Today, Morse code is still used by radio enthusiasts and remembered as one of the first forms of fast long-distance communication.

Morse code uses dots and dashes to represent letters and numbers. Short signals are called "dots" and long signals are called "dashes".
